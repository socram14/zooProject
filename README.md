# 🐾 Zoo Management System  
A console-based zoo simulation built with Java. This project focuses on applying core Object-Oriented Programming (OOP) principles such as inheritance, interfaces, and polymorphism.


## ✨ Features  
- Animal Hierarchy: Base `Animal` class extended by `Tiger`, `Dolphin`, and `Penguin`  
- Interface-Based Design: `Eat`, `Walk`, and `Swim` behaviors implemented where applicable  
- Polymorphism: Method overriding across different animal types  
- Multi-Behavior Class: Penguin supports both walking and swimming  
- Interactive Menu: View and trigger animal behaviors through user input  


## 🛠️ Technologies  
- Language: Java  
- Core Concepts:  
  - Classes & Objects  
  - Inheritance  
  - Abstract Classes  
  - Interfaces (multiple implementation)  
  - Polymorphism
